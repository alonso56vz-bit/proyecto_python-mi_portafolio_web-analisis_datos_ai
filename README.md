# proyecto_python-mi_portafolio_web-analisis_datos_ai
Participación en el curso bootcamp sobre ciencia de datos e IA
Utilizamos diferentes plataformas con Google sheets, Google colabs, power Bi y Geminini integrada a Googlesheets 
cómo estudiante fui recorrido un camino de aprendizaje en el cual comencé aprendiendo el que era la ciencia de datos y diversos conceptos de esta utilizando power bi para generar diversas tablas con una base de datos proporcionada luego use Google colab para introducción a python y uso de la biblioteca pandas, finalizando con el usu de géminis para facilitar el análisis de dato apoyádo de Google shets
